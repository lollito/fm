# Football Manager Product Overview

## What is this?
A web-based Football Manager game that simulates running a football club. Players can manage teams, handle transfers, set formations, and compete in leagues.

## Architecture
- **Backend**: Spring Boot REST API with JWT authentication
- **Frontend**: Two React applications (main game + admin panel)
- **Database**: H2 (development) / MySQL (production)

## Key Features
- Team management and player transfers
- Match simulation and scheduling
- League rankings and statistics
- User authentication and authorization
- Stadium and financial management
- Formation and tactical setup

## Target Users
- Football/soccer enthusiasts who enjoy management simulation games
- Players who want to experience running a football club online
- Administrators who need to manage the game system

## Business Logic
The game simulates football club management including player stats, match outcomes, financial aspects, and league progression. The system handles automated match processing and maintains historical data for comprehensive gameplay experience.
